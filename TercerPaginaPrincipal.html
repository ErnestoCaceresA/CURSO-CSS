CONCEPTOS BASICOS DE GRID


GRID ES UNA PROPIEDADE DE DISPLAY Y SIRVE PARA TRAbAJAR CON GRILLAS EN EL LOYOUT





--------------------------------------------------------------------

Grid container


Es como flex content, al darle display grid al CONTENEDOR este no cambia sigue siendo un bloque, lo que cambia son los items de adentro


--------------------------------------------------------------------


--------------------------------------------------------------------

Grid item


Son todos los elementos que estan dentro del grid (igual solo los hijos directos no nietos)


--------------------------------------------------------------------


--------------------------------------------------------------------

Grid cell


son las celdas de el grid


--------------------------------------------------------------------


--------------------------------------------------------------------

Grid Tracks (Column y Row)


son por asi decirlo las filas y columnas de un grid, solamente filas y columndas


--------------------------------------------------------------------


--------------------------------------------------------------------

Grid Area


las areas las definimos nosotros


--------------------------------------------------------------------


--------------------------------------------------------------------

Grid Line (Column line y Rom Line)

propiedades para poner al CONTENEDOR grid que afectan a los items:




--------------------------------------------------------------------

grid-template-rows


Es para decir cuantas filas va a tener el grid

SE PONE grid-template-rows: 150px 150px 150px (OSEA SE PONE LA MEDIDA DEL ALTO DE LA FILA Y SE PONE TANTAS QUIERAS PARA DETERMINAR EL NUMERO DE FILAS)


EN ESTE CASO EN EL GRID HAY SOLAMENTE 3 FILAS QUE PROGRAMAMOS (GRID EXPLICITO), LAS DEMAS ETIQUETAS QUE NO ESTAN PROGRAMADAS PARA EL GRID SE CONSIDERAN (GRID IMPLICITO)

AUTO-FILL


ES COMO EL WRAP DE FLEXBOX

una forma de hacer que CRESCA DINAMICAMENTE Esto lo que hace es que va agregando o quitando columnas y/o filas cada que se modifica las resoluciones

se pone dentro del REPEAT EN VEZ DE PONER LA CANTIDAD EN LA QUE SE VA A REPETIR LAS MEDIDAS DE LAS COLUMNA O FILA SE LE PONE AUTO-FILL

BASICAMENTE LO QUE DICE AQUI ES, SI HAY ESPACIO EN ESTE CASO EN EL ANCHO DE LA PANTALLA, PARA QUE ENTRE UNA CAJA (QUE CUMPLA CON LAS PROPIEDADES MINMAX) DADAS DE ABAJO LA METE, SI NO HAY ESPACIO (QUE CUMPLA CON LAS PROPIEDADES DADAS DE MINMAX) ENTONCES HECHA PARA ABAJO LAS CAJAS


--------------------------------------------------------------------

grid-template-rows: repeat(3,150px);
grid-template-columns:repeat(auto-fill, minmax(120px, 1fr));
