ANIMACIONES


--------------------------------------------------------------------

KEYFRAMES = para ver en que porcentaje de la animacion hace cada cambio (DEBE ASIGNAR NOMRE A ANIMACION)

ANIMATION-NAME = para hacer llamar a la animacion creada con los keyframes al elemento que queramos con el nombre asignado

ANIMATION-DURATION = duracion de la animacion

ANIMATION-TIMING-FUNCTION = igual que lo que se vio en transiciones para ver que tipo poner

ANIMATION-ITERATION-COUNT = sirve para poner cuantas veces quieres que se repita la animacion (SE PUEDE PONER "INFINITE")

ANIMATION-DIRECTION = (NORMAL: va de 0% a 100%, REVERSE: va de 100% a 0%, ALTERNATE: va de 0 a 100% y despues de 100% a 0%, ALTERNATE-REVERSE)

ANIMATION-FILL-MODE = define como se quedara el elemento cuando termine la animacion (NONE: queda como estaba al principio, FORWARDS: queda como finaliza el la animacion osea que si da 3 veces la animacion, cuando acabe se quedara como exactamente quedo en esa tercera vez)


--------------------------------------------------------------------

animacionContainer .caja{
background: #025;
width: 130px;
height: 130px;
margin: 10px 15px;
position: relative;
animation-name: NombreTransicion;
animation-duration: 4.3s;
animation-timing-function: ease;
animation-iteration-count: infinite;
animation-direction: alternate;
}

@keyframes NombreTransicion{
0%{
left: 0;
background: #025;
}
50%{
background: rgb(168, 87, 62);
left: 90%;
}
100%{
left: 0%;
background: rgb(32, 104, 32);
}
}

CUBIC-BEZIER


--------------------------------------------------------------------

esta es una propiedad para el ANIMATION-TIMING-FUNCTION que sirve para modificar manualmente la curva de la animacion o transcion y poder hacer cosas super cool CON ESTO SE PUEDEN INCLUSO CREAR EFECTOS REALES DE REBOTE Y TODO


--------------------------------------------------------------------

PARA PODER HACERLO CON EL EXPLORADOR, DAS CLICK EN INSPECCIONAR AL ELEMENTO QUE TIENE EL "ANIMATION-TIMING-FUNCTION: CUBIC-BEZIER()" Y EN ESTILO TE SALDRA PARA MODIFICAR GRAFICAMENTE ESA CURVA E IR VIENDO COMO SE VERIA, Y SOLO PUES COPIAS LAS COORDENADAS DEL CUBIC-BEZIER Y YA









animation-timing-function: cubic-bezier(.83,-0.41,.01,1.72);
